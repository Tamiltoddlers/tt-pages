Uyir Ezhuthukkal

I've shared all the activities that I did with my son to learn Tamil Uyir ezhuthukkal. Keep all the things ready with you before day so that you will be able to finish the activity. Take things slowly and start teaching one letter a week. Do not assume that Tamil is a difficult language to learn. Never say that in front of the child. If we practise it is going to be very easy. So encourage them even if they make mistakes. Reminding you once again, Take it slowly one letter a week !  

All the activities to be done under parent guidance only.
